# Labs: Сборник мини-проектов

Добро пожаловать в репозиторий **Labs**! 🚀

Это коллекция мини-проектов, созданных разработчиком-преподавателем для демонстрации различных концепций веб-разработки, архитектурных паттернов и современных технологий для любимых учеников и замечательных коллег.

## 🛠 Технологический стек

В этом репозитории вы найдете проекты, использующие:

- **Frontend**: HTML5, CSS3 (Vanilla, Modern Layouts), JavaScript (ES6+)
- **Frameworks**: [Vue.js 3](https://vuejs.org/) (Composition API, Vite, Pinia)
- **Styling**: CSS Modules, Tailwind CSS
- **Tools**: Vite, Bun, NPM, PNPM
